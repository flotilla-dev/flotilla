## Quick Setup Script

After copying all files, run this to verify structure:

```bash
#!/bin/bash
# verify_structure.sh

echo "Verifying file structure..."

files=(
    "requirements.txt"
    "README.md"
    "main.py"
    "agents/fabric_data_agent.py"
    "agents/decisioning_agent.py"
    "agents/orchestration_agent.py"
    "agents/business_logic/base_business_agent.py"
    "agents/business_logic/agent_registry.py"
    "agents/business_logic/pricing_agent.py"
    "agents/business_logic/inventory_agent.py"
    "clients/block_mcp_client.py"
    "config/config_loader.py"
    "models/config_models.py"
    "utils/logger.py"
    "workflows/pricing_optimization.json"
)

missing=0
for file in "${files[@]}"; do
    if [ ! -f "$file" ]; then
        echo "❌ Missing: $file"
        missing=$((missing + 1))
    else
        echo "✅ Found: $file"
    fi
done

if [ $missing -eq 0 ]; then
    echo ""
    echo "🎉 All files present!"
    echo "Next steps:"
    echo "1. pip install -r requirements.txt"
    echo "2. python main.py init"
    echo "3. Update config files with your credentials"
else
    echo ""
    echo "⚠️  $missing files missing"
fi
```

Save as `verify_structure.sh`, make executable with `chmod +x verify_structure.sh`, and run with `./verify_structure.sh`.
